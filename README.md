### Installation Guide:

* ```$ git clone: https://github.com/Viltorn/frontend-project-12.git```
* ```$ make install```
* ```$ make start``` - run the frontend & backend
* ```$ make start-frontend``` - run only frontend
* ```$ make start-backend``` - run only backend
